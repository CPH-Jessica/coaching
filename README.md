# The Invisible Pen — Coaching

Intake forms for 1:1 Voice Calibration Coaching with Jessica (The Invisible Pen / Clark Publishing House).

**Established authors:** https://cph-jessica.github.io/coaching/
**Newer authors:** https://cph-jessica.github.io/coaching/new-authors/

Both "Author Snapshot" forms are fully self-contained: they collect no data on any server. When an applicant submits, their answers are bundled into text on their own device and they email them to clarkpublishinghouse@gmail.com. Nothing is stored online.
